**Pathological Gait Classification using a Hybrid CNN-LSTM — Team Lead, Developer — 2024**
Trained a Hybrid CNN-LSTM model that classifies 5 different pathological gaits (Antalgic, Lurch, Steppage, Stiff-legged, Trendelenburg) and a normal gait. Features used are the joint angles of the hip, knee, and ankle (900 frames per walk).
* Architecture: Joint angles (hip, knee, ankle) across 900 frames → OpenPose for 2D pose estimation and joint angle calculation → CNN for spatial feature extraction → LSTM for temporal sequence learning → 6-class classification
* Results: 99.72% training accuracy, 66.67% testing accuracy
* Technologies: Python, Deep Learning (Keras), Computer Vision (OpenPose)
